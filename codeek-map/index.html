Codeek Map

Codeek Map is a practical map tool for saving important places, organizing route ideas, and keeping field notes close to the map in a clean, map-first workflow.

What It Does

The app helps users capture locations, attach notes and photos, draw routes, outline areas, and export useful map data when needed.

  • Save markers with titles, notes, and coordinates
  • Attach photos to saved markers
  • Draw routes with waypoint support
  • Outline areas directly on the map
  • Search by place name, address, or coordinates
  • Export marker and route data in CSV or JSON

App Details

  • Local-first storage
  • Built for planning, scouting, and field notes
  • No account required for core features
  • Part of the Codeek app portfolio
