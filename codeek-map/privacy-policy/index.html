Codeek Map Privacy Policy

Last updated: April 4, 2026

App page All policies

Codeek Map is a mobile map tool that lets users save markers, attach photos, draw routes and areas, search places, and export selected data. This Privacy Policy explains what information the app handles, how that information is used, and what choices users have.

Summary

  • Codeek Map does not require account registration
  • Codeek Map does not include ads, analytics SDKs, or crash reporting in the current app build
  • Most app data is stored locally on the user's device
  • Location and photo library access are optional and used only for features the user chooses to use
  • Place search sends the user's search query to the OpenStreetMap Nominatim search service so results can be returned

Information We Handle

  • Foreground device location for current-position and recenter features
  • Photos selected by the user and copied into the app's local storage so they remain linked to saved marker records
  • User-created map content such as markers, coordinates, notes, attached photo references, routes, waypoints, area geometry, preferences, and recent searches
  • Search queries sent to https://nominatim.openstreetmap.org/ when place or address search is used

How Information Is Used

  • Showing and recentering the map
  • Saving user-created map items
  • Attaching selected photos to markers
  • Returning place search results
  • Exporting map data to CSV or JSON files
  • Opening the device share sheet when the user chooses to share an export

Data Storage

Most data created in the app is stored locally on the user's device, including saved markers, routes, areas, preferences, and recent searches. Marker photos selected by the user are also stored in the app's local file storage on the device.

Data Sharing

Codeek Map does not sell personal information and does not share user data with advertising networks. Data may be shared only when search queries are sent to OpenStreetMap Nominatim or when export files are shared because the user explicitly chooses to share them.

Permissions

  • ACCESS_FINE_LOCATION and ACCESS_COARSE_LOCATION for current-location features
  • READ_MEDIA_IMAGES or READ_EXTERNAL_STORAGE to let the user pick photos from the device library

Data Retention and Deletion

Because most app data is stored locally on the device, users can remove data by deleting saved items inside the app, clearing app storage through device settings, or uninstalling the app.

Children's Privacy

Codeek Map is not intended to knowingly collect personal information from children.

Security

No method of electronic storage or transmission is completely secure, so absolute security cannot be guaranteed.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. The updated version should be published with a revised effective date.
