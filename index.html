Welcome to the official developer website for codeek. This page is ready to be used as the developer website URL for Google Play and AdMob.

To complete Google Play verification:

  • Set this site as your Developer website in Google Play Console.
  • Make sure /app-ads.txt is available at the root of the published website.

Publishing notes

If you publish this repository with GitHub Pages, your site will be available at:

https://codeek-ai.github.io/github.io/

For Google Play, a better option is a root-domain URL such as a custom domain or using a repository named codeek-ai.github.io.

Once your website is live, verify https://your-site.com/app-ads.txt in a browser.
