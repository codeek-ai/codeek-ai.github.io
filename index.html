Google Play And AdMob Verification

This website is the shared public developer website for the app portfolio. To allow Google Play and AdMob to verify app listings that use this site, keep the same root website URL in each Play Console listing that should point here:

https://codeek-ai.github.io/

  • Set https://codeek-ai.github.io/ as the developer website in each relevant Google Play app listing.
  • Make sure https://codeek-ai.github.io/app-ads.txt loads publicly.
  • The published app-ads.txt file is hosted at the root of this website for verification checks.
